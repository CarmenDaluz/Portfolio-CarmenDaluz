# 🖥️ Portfolio — Carmen Daluz
![Captura del portfolio](assets/screenshot.png)
Bienvenida a mi portfolio personal como desarrolladora web frontend. Este proyecto es más que una web de presentación: es un espacio donde la estética y el código se dan la mano.

🔗 **[Ver portfolio en vivo →](https://carmendaluz.netlify.app)**

---

## 🧩 ¿Qué encontrarás aquí?

El portfolio está dividido en las siguientes secciones:

- **Home** — Presentación con animación typewriter y efecto de partículas
- **About Me** — Cuatro tarjetas narrativas que cuentan quién soy, de dónde vengo y hacia dónde voy
- **Projects** — Proyectos destacados con descripción, tecnologías y links
- **Contact** — Contacto y redes profesionales

---

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|---|---|
| React + Vite | Framework y bundler principal |
| JavaScript (ES6+) | Lógica e interactividad |
| CSS3 + Variables CSS | Estilos con sistema de diseño propio |
| React Router DOM | Navegación entre secciones |
| @tsparticles | Animación de partículas en el fondo |
| GitHub Pages / Netlify | Despliegue |

## 🚀 Cómo ejecutarlo en local

```bash
# 1. Clona el repositorio
git clone https://github.com/CarmenDaluz/Portfolio-CarmenDaluz.git

# 2. Entra en la carpeta del proyecto
cd Portfolio-CarmenDaluz

# 3. Instala las dependencias
npm install

# 4. Arranca el servidor de desarrollo
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
